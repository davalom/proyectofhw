Clonezilla

Clonezilla es un programa de creación/restauración de imágenes de particiones y discos completos. Es útil para desplegar sistemas de forma rápida en varios ordenadores y realizar copias de seguridad de sistemas completos.

En estas prácticas vamos a usar Clonezilla live, la versión diseñada para crear imágenes de sistemas individuales

Tiene licencia GNU GPLv2 (Open Source) aunque para su funcionamiento depende de más software (también Open Source) con otros tipos de licencia.

A continuación se listan algunas de sus características principales:

  • Soporte para gran cantidad de sistemas de archivos.
  • Soporte para volúmenes virtuales LVM2 en sistemas Linux.
  • Reinstalación del boot loader.
  • Soporte para discos MBR y GPT.
  • Modo desatendido.
  • Soporte para restaurar varios dispositivos simultáneamente desde una única imagen.
  • Soporte para encriptación de imágenes.
  • Soporte para copias multicast y protocolo Bittorrent.
  • Capacidad para restaurar desde dispositivo local, servidor ssh, samba NFS o WebDAV.

Los requisitos mínimos para su funcionamiento son:

  • Procesador de 32 o 64 bits.
  • 196MB de memoria RAM.
  • Dispositivo de arranque como lector de CD o puerto USB.
  • Disco de destino con espacio suficiente para realizar la copia.

Primeros pasos

Crear USB de arranque

Antes de empezar a utilizar Clonezilla tendremos que crear un medio arrancable desde el que ejecutar la aplicación. Para ambas prácticas utilizaremos un pendrive de 64GB como pendrive de arranque, aunque se pueden utilizar otros medios como puede ser un CD o DVD.

A continución se detallan los pasos a seguir para crearlo. Este procedimiento es idéntico tanto si queremos hacer una copia en un disco local como en un servidor remoto y sólo es necesario hacerlo una vez.

#01. El primer paso es descargarse la versión actual de Clonezilla, la cual podemos obtener en el siguiente enlace:

Web oficial de Clonezilla

Para la práctica descargaremos la versión stable 2.8.1 en su variante x86 y en formato ISO.

#02. También necesitaremos el programa Rufus o similar para crear el USB arrancable. Lo podemos descargar desde su web oficial:

Web oficial de Rufus

Descargaremos la versión portable.

#03. Intoduciomos un pendrive vacío en el ordenador y lo seleccionamos en la interfaz de Rufus.

Selecionamos la imagen ISO de Clonezilla y escribimos un nombre de etiqueta para identificar el pendrive.

Pulsamos en el botón «empezar»

#04. La imagen de Clonezilla que hemos descargado puede ser escrita de dos modos distintos. Rufus nos recomienda utilizar el modo Imagen ISO, por lo que seleccionamos la opción «Escribir en modo Imagen ISO (Recomendado)» y le damos a «OK».

#05. Ahora nos pregunta si le damos permiso al programa para conectarse a internet y descargar unos ficheros adicionales que necesita para completar el proceso. Pulsamos en «Sí».

Por último, nos recordará que todos los datos contenidos en el pendrive serán borrados. Si estamos seguros de que no hay guardado nada importante, le damos a «Aceptar».

#06. Una vez finalizado el proceso, le damos al botón «CERRAR» y ya tendremos listo un pendrive USB arrancable con Clonezilla.

Podemos pasar a la siguiente sección «Configuración inicial».

Configuración inicial

Esta parte es común a las dos prácticas y consiste principalmente en arrancar el ordenador (del que queremos hacer/restaurar la imagen) desde el pendrive de Clonezilla.

También se harán las primeras configuraciones como la selección del idioma y distribución de teclado. A continuación se detallan los pasos seguidos:

#08. Tras arrancar el ordenador desde la memoria USB aparecerá la pantalla de bienvenida de Clonezilla.

De las opciones que nos da, podemos escoger la primera o segunda, aunque la segunda es preferible ya que nos permite trabajar sin necesidad de tener el pendrive conectado.

#09. Tras cargar el programa en la memoria RAM aparecerá el menú de selección de idioma. Elegimos el idioma con el que queramos trabajar, a ser posible el idioma del teclado de la máquina para facilitar el siguiente paso. En estas prácticas se han utilizado indistintamente los idiomas Español e Inglés.

#10. El siguiente paso es seleccionar la distribución de teclado. El sistema para cambiar la distrubución es bastante confuso, por lo que es mejor dejar la distribución por defecto, teniendo en cuenta que algunas teclas pueden cambiar de posición.

#11. Seleccionamos iniciar Clonezilla. La otra opción es para iniciar el programa en línea de comandos, útil si se desea trabajar de forma desatendida.

#12. Seleccionamos disposivo-imagen, ya que en ambas prácticas queremos crear una imagen del disco interno del ordenador. En el siguiente paso deberemos decidir si queremos crear/restaurar la copia desde un dispositivo local o desde un servidor remoto, continuando en la sección correspondiente.

Restaurar imagen

Test Disk se distruibuye en forma portable. Esto quiere decir que se puede usar sin necesidad de instalarlo. Si queremos tenerlo en el equipo como si fuera una versión instlada simplemente tenemos que descomprimir los ficheros en un lugar de nuestra elección (en mi caso he creado una carpeta «programas portables» en C:) y guardar un acceso directo al ejecutable en la siguiente ruta:

C:\Users\[$USUARIO]\AppData\Roaming\Microsoft\Windows\Start Menu\Programs

De esta forma Windows lo añade ai índice del menú de inicio y permitiendo buscarlo desde allí.
