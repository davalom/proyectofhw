DRBL / Clonezilla

Clonezilla es un programa de creación/restauración de imágenes de particiones y discos completos. Es útil para desplegar sistemas de forma rápida en varios ordenadores y realizar copias de seguridad de sistemas completos.

En estas prácticas vamos a usar Clonezilla live, la versión diseñada para crear imágenes de sistemas individuales

Tiene licencia GNU GPLv2 (Open Source) aunque para su funcionamiento depende de más software (también Open Source) con otros tipos de licencia.

A continuación se listan algunas de sus características principales:

  • Soporte para gran cantidad de sistemas de archivos.
  • Soporte para volúmenes virtuales LVM2 en sistemas Linux.
  • Reinstalación del boot loader.
  • Soporte para discos MBR y GPT.
  • Modo desatendido.
  • Soporte para restaurar varios dispositivos simultáneamente desde una única imagen.
  • Soporte para encriptación de imágenes.
  • Soporte para copias multicast y protocolo Bittorrent.
  • Capacidad para restaurar desde dispositivo local, servidor ssh, samba NFS o WebDAV.

Los requisitos mínimos para su funcionamiento son:

  • Procesador de 32 o 64 bits.
  • 196MB de memoria RAM.
  • Dispositivo de arranque como lector de CD o puerto USB.
  • Disco de destino con espacio suficiente para realizar la copia.
